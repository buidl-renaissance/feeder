## Source Types

The feeder supports the following content source types:

| Type | Description | Documentation |
|------|-------------|---------------|
| RSS | RSS/Atom feeds | Built-in |
| YouTube | YouTube channel feeds | Built-in |
| API | Generic API endpoints | Built-in |
| File | File uploads | Built-in |
| **Luma** | Lu.ma event platform | [Event Sources](event-sources.md) |
| **Meetup** | Meetup.com events | [Event Sources](event-sources.md) |
| **RA** | Resident Advisor events | [Event Sources](event-sources.md) |
